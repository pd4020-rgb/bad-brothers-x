# JUNGKING 홈페이지 전시 아카이브

`www.jungking.com`의 EXHIBITION 메뉴와 각 상세 페이지를 기준으로 정리한 로컬 아카이브입니다.

- 전시 폴더: 25개
- 다운로드한 원본 이미지: 133개
- 전체 용량: 약 130MB
- 각 폴더의 `README.md`: 전시 제목, 홈페이지 주소, 전시 설명
- `manifest.json`: 전시와 Wix 원본 이미지의 연결 정보
- `_source/wix_media_inventory.json`: Wix 미디어 관리자에서 확인한 전체 파일 인벤토리

## 확인 필요

`06_2023 Collabo Collage Gallery,2인전,울산`의 홈페이지 주소는 현재 2025 갤러리오로라 페이지로 연결됩니다. 잘못된 이미지를 넣지 않기 위해 해당 폴더에는 설명과 경고만 남기고 이미지 분류를 보류했습니다.

## 정리 기준

Wix 미디어 관리자에 저장된 모든 파일을 복제한 것이 아니라, 홈페이지의 각 전시 페이지에 실제로 사용된 이미지 원본만 전시별로 내려받았습니다. 같은 이미지가 여러 전시에 사용된 경우에는 각 전시 폴더에 각각 보존했습니다.
