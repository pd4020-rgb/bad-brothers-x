# AGENTS.md — AI 에이전트 공용 규칙

## 작업 시작 전 필수 읽기
1. `PROJECT_BIBLE.md` — 프로젝트가 뭔지, 왜 만드는지
2. `HANDOFF.md` — 직전 AI가 뭘 했고, 다음에 뭘 해야 하는지
3. `git log --oneline -5` — 최근 코드 변경 내역 확인

## 작업 완료 시 필수 수행
1. `HANDOFF.md` 업데이트 (완료된 것 체크, 다음 할 일 기술)
2. `git add -A && git commit -m "설명"` — 복구 지점 생성
3. 배포했으면 배포 URL과 시간 기록

## 금지사항
- 대표님에게 프로젝트 설명을 다시 요구하지 말 것
- 플레이스홀더(회색 박스) 사용 금지 — 실제 이미지 사용
- "딸깍이" 관련 표현 금지
- 사이트 텍스트를 한국어로 바꾸지 말 것 (글로벌 타겟)
