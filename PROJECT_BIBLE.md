## 🚨 핵심 비전 (절대 잊지 말 것 — 수십 번 말한 내용)

**BAD BROTHERS X = AI 엔터테인먼트 레이블/회사**

### 이게 무엇인가
- **Big Hit Entertainment(HYBE)의 AI 버전**
- **배트맨, 슈퍼맨, 마리오처럼** — 가상 캐릭터지만 얼굴/나이/직업/스토리/음악이 있어 "살아있음"
- **Gorillaz, 버추얼 K-pop 아이돌** 처럼 가상 AI 아티스트들이 실제 음악과 콘텐츠를 냄

- 다양한 장르, 다양한 캐릭터 → 하나의 온라인 엔터테인먼트 생태계

## 🔥 왜 이 프로젝트가 존재하는가 (창립 배경 — 절대 잊지 말 것)

### 정킹이 직접 말한 이유
정킹은 한국에서 **15년간 수십 점의 작품을 전시했지만 단 5점**밖에 팔지 못했다.
이유:
- 갤러리가 전시를 열어주지 않으면 작가는 아무것도 할 수 없다
- 갤러리는 판매 수익의 **50%를 가져간다**
- 전시 중에는 작품을 **3개월 이상 갤러리가 보관**하며 작가 본인이 팔지 못하게 막는다
- 가수도 마찬가지 — 무대에 설 수 있는 기회 자체가 없다
- 결국 **자본을 가진 자(갤러리, 기획사, 대형 엔터)가 기술과 무대를 독점**해왔다

### AI가 바꾼 것
> "이전에는 자본이 있는 자가 기술을 독점해서 시각화했다면, 이제 인공지능으로 누구나 — 또는 원천기술·상상력·아이디어가 있으면 — 기술을 커스텀화해서 세상에 나올 수 있다."
> — JUNGKING

- 픽사가 마리오를 영화로 만들려면 수천억이 필요했다
- 닌텐도가 포켓몬을 게임으로 만들려면 수백명의 개발자가 필요했다
- **지금은 정킹의 오리지널 스케치 하나가 AI를 통해 움직이고, 노래하고, 영상이 된다**

### 닌텐도 모델 = BBX 모델
```
닌텐도 (플랫폼)        → BAD BROTHERS X (AI 엔터 레이블)
마리오 / 포켓몬        → CHARACTER X / 88X / 미래 캐릭터들
게임 → 굿즈 → 영화    → 음악 → AI 영상 → 프린트 → 오리지널 원화
각 IP가 서로 홍보      → 선순환(Virtuous Cycle) 생태계
```

**닌텐도가 마리오를 만들었을 때 마리오는 픽셀이었다. 하지만 세계관이 있었다.**
**정킹의 스케치가 그 픽셀이다. BBX가 닌텐도다.**

### 수익 모델 (재고 없이, 갤러리 없이, 전 세계)
| 레벨 | 상품 | 플랫폼 | 마진 |
|---|---|---|---|
| **입문** | 디지털 아트팩, 음원 다운로드 | Gumroad | 100% |
| **미드** | 프리미엄 아트 프린트, 에디션 | Printful | 50–60% |
| **하이엔드** | 에디션 원화 (넘버링+서명+인증서) | 직접 문의 | **100%** |
| **IP 라이선스** | 브랜드 협업, 광고 캐릭터 사용 | 직접 계약 | 100% |

**선순환 구조:**
`오리지널 아트 → 캐릭터 세계관 → AI 음악/영상 → 굿즈/에디션 → 원화 판매`
`→ 각 콘텐츠가 서로를 홍보 → 팬이 생기고 → 다시 새 캐릭터 출시`

### 핵심 원칙
1. 캐릭터는 가상이지만 **세계관은 진짜**여야 함
2. 각 캐릭터는 얼굴, 나이, 직업, 스토리, 음악 장르가 있어야 함
3. 정킹 본인도 BBX 소속 아티스트로 포지셔닝 (창립자 + 아티스트)
4. 사이트 = 레이블 공식 포털 (정킹 개인 포트폴리오 사이트가 아님)
5. ROSTER = 소속 아티스트 라인업 (레이블 회사처럼)
6. **갤러리 없이, 기획사 없이, 재고 없이 — 직접 글로벌 판매가 목표**
7. 미국 시장 먼저 → 성공하면 한국도 따라온다

## 2. BAD BROTHERS X가 뭔지

### 한 줄 정의
**AI 엔터테인먼트 레이블 — 가상 AI 캐릭터 아티스트들이 실제 음악/영상/콘텐츠를 내는 온라인 엔터테인먼트 회사.**

### 비유
- **빅히트(HYBE) 같은 엔터테인먼트 회사의 AI 버전**
- 정킹의 개인 홈페이지가 **아님** — 회사/레이블임
- 정킹은 BBX에 "입점한 아티스트"로 포지셔닝

### 구조
```
BAD BROTHERS X (AI 엔터 에이전시)
├── JUNGKING — 원작자/Creative Director (40년 아날로그 팝아트)
├── CHARACTER X — 녹색 물고기 캐릭터 (가상 아티스트/퍼포머)
├── 88X — AI 음악 프로듀서 엔진
└── 결과물: MV, 음악, 브랜드 필름, 굿즈
```

### 핵심 철학 (절대 변경하지 말 것)
> "This is not AI replacing the artist. This is AI extending an original creative universe."
> "이건 AI가 만든 게 아니라, 40년 예술가의 오리지널 비전을 AI가 실현한 것이다."

- ❌ "딸깍이" (AI 자동 생성물) 취급 절대 금지

### 🚨 공통 주의사항 (모든 모드)

1. **배경색은 항상 순수 검정 `#000000`** — 녹색/회색 배경 노출 금지
2. **이미지는 `object-fit: cover`** — 찌그러지거나 비율 깨짐 금지
3. **글씨 크기 통일감** — 같은 유형의 텍스트는 동일 사이즈 (본문 0.95rem, 상태뱃지 0.65~0.75rem)
4. **가로 모드에서 이미지 잘림 확인** — aspect-ratio가 적절한지 반드시 확인
5. **메뉴 열었을 때 뒤 콘텐츠 보여야 함** — 풀스크린 오버레이 금지, 70% 드로어 + 반투명 배경
6. **번역 방지** — `<html lang="en">`, `<meta name="google" content="notranslate">` 유지
7. **PWA 아이콘** — 검정 배경 + 흰색 영문 B (icon-192.png, icon-512.png)
8. **타이포그래피** — 메인 제목과 본문·아카이브·가격 모두 `Inter` 계열로 통일. 제목은 700~900, 본문은 300~400, 메타 정보는 600 + 자간으로 위계만 구분
9. **색상 절제** — 흰색·회색이 기본이며 녹색은 작은 상태 표시나 포인트에만 사용. 가격과 긴 텍스트에 강한 녹색 사용 금지
10. **파티클 인트로** — 데스크톱은 문장 왼쪽/얼굴 오른쪽, 모바일은 문장 위/얼굴 오른쪽 아래로 분리해 서로 가리지 않게 유지

```
1. PC 브라우저에서 전체 스크롤 → 각 섹션 레이아웃 확인

6. 스크롤-투-탑 버튼 동작 확인
7. 모든 텍스트가 영문인지 확인 (한글 노출 금지)
```
